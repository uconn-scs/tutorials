UConn SCS Tutorials

This site hosts practical tutorials, examples, and tools developed by Statistical Consulting Services (SCS) to support reproducible data analysis and research workflows.

Materials range from short, focused guides (e.g., implementing logistic regression in R) to interactive tools (e.g., Shiny applications for urn randomization). Most resources are hosted on GitHub Pages and are freely available for learning, teaching, and adaptation.

These materials are intended to complement—rather than replace—consultation or collaboration with SCS.
